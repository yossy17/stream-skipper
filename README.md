## Overview

**Stream Speeder**は、Netflix, Prime Video でイントロとエンディングをスキップできます｡

## Features

| Key |          Description           |
| :-: | :----------------------------: |
|  z  |     スキップ機能の切り替え     |
|  x  |   イントロスキップの切り替え   |
|  c  | エンディングスキップの切り替え |
|  n  |         次のエピソード         |
